## Phase 1: 프로토타입 (Prototype)

> 핵심 플레이 루프가 "재미있는가"를 검증

### 1-1. 코어 메카닉
- [x] 탑다운 4방향 이동 + 카메라 시스템 (전투 시 고정/추적)
- [x] NPC 대화 시스템 (하단 텍스트박스, 선택지 분기, 조건부 대사)
- [x] 심볼 인카운터 + 필드 전투 (공격/구르기/패링/반격기)
- [x] 맵 전환 (문 오브젝트 + F키 + 페이드 효과)

### 1-2. 그레이박스 레벨
- [x] 마을 테스트맵 (1280×720, 벽 경계, 플레이스홀더 그래픽)
- [x] 교역도시 테스트맵 (NPC 4명 — 쌍눈 2 + 외눈 2)
- [x] NPC 6명 배치 (조건부 대사, 평판 연동)
- [x] 적대NPC 2종 (대화 → 전투 전환, 구경꾼 시스템)

### 1-3. 시스템 기초
- [x] 평판 시스템 (oneeye/twoeye/trade, 5단계, 조건부 대사 연동)
- [x] 사건 플래그 시스템 (대화 선택지 → 플래그 설정 → 분기)
- [x] 세이브/로드 (ESC 메뉴, JSON 1슬롯, 위치/체력/플래그/평판 보존)
- [x] 일시정지 메뉴 (ESC → 계속/저장/불러오기)

### ✅ 완료 조건
마을을 걸어다니고, NPC와 대화하고, 전투하고, 선택에 따라 NPC 반응이 바뀌는 플레이어블 빌드.

## Phase 2: 버티컬 슬라이스 (Vertical Slice)

> Ch0~Ch1을 완성도 있게 만들어 "이 게임의 맛"을 보여주는 구간

### 2-1. Ch0 — 외눈박이 마을 완성
- [ ] 아트 적용 (수채화/파스텔 톤)
- [ ] NPC 대사 작성 + 연출
- [ ] 복선 오브젝트 배치 (석상, 촌장 가면 등)
- [ ] BGM/SE 배치

### 2-2. Ch1 — 쌍눈박이 마을 구현
- [ ] 맵 제작 + 아트 (차가운 고딕 톤)
- [ ] 차별 체험 이벤트 구현
- [ ] 전투 빈도 조절 시스템
- [ ] 루메 등장 + 거부/수용 시스템 (3단계)

### 2-3. 루메 동행 시스템
- [ ] 루메 AI (따라오기, 고유 반응)
- [ ] 루메 유/무에 따른 NPC 반응 분기
- [ ] 안대 아이템 이벤트

### ✅ 완료 조건
Ch0→Ch1을 처음부터 끝까지 플레이 가능. 루메 수용/거부 분기 작동. 감정 곡선 체감 가능.

## Phase 3: 콘텐츠 완성 (Full Content)

> 전 챕터 + 전 엔딩 구현

### 3-1. Ch2 — 교역 도시
- [ ] 맵 (다채로운 항구 도시)
- [ ] 공존 → 붕괴 이벤트 체인
- [ ] 결정적 복선 집중 배치
- [ ] 동료 젊은이 재조우

### 3-2. Ch3 — 유물 여정 + 발동
- [ ] 유물 탐색 퍼즐/여정
- [ ] 유물 발동 연출
- [ ] 발동 후 세계 변화 (색감 전환, NPC 대사 전면 교체)

### 3-3. 에필로그
- [ ] 마을 귀환 + 변화된 세계 탐색
- [ ] 촌장 가면 해제 씬
- [ ] 엔딩 분기 (진/일반/몰살/회피)

### 3-4. 다회차 시스템
- [ ] 기억의 조각 수집 시스템
- [ ] 회차별 NPC 반응/환경 변화 (레이어 1~5)
- [ ] 히든 엔딩 개방 조건

### 3-5. NPC 반응 시스템 확장
- [ ] 관계 기억 (핵심 NPC 10~15명)
- [ ] 군중 분위기 연출
- [ ] 루메 고유 반응 레이어

### ✅ 완료 조건
5개 엔딩 전부 도달 가능. 전 챕터 플레이 가능. 10-12시간 분량 확보.

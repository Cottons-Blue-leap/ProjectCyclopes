## 2. 핵심 메시지 (삼단 구조)

| 단계 | 메시지 | 게임 내 대응 |
|------|--------|-------------|
| **A. 시작** | 당연한 건 없다 — 네가 서 있는 곳이 달라지면 세상도 달라진다 | 유물 발동으로 당연했던 세상이 뒤집힘 |
| **B. 과정** | 다르다는 건 틀린 게 아니라, 아직 서로의 자리에 서보지 않았다는 뜻이다 | 쌍눈박이가 차별을 체험 |
| **C. 결말** | 그럼에도, 너는 그 자체로 충분하다 | 외견이 바뀌어도 나는 나라는 깨달음 |

**주제의식:** 세대와 세대별로 과거의 기록을 숨기려 드는 세상에서, 차별의 세계를 해결할 수 있는 것은 개개인의 차별에 대한 의문이다.

**발상 원점:** "외눈박이 마을에 간 정상인은 비정상 취급을 받는다"는 고전적 우화 + 미디어커뮤니케이션 전공(문화계발효과 → 필터 버블 연결)

## 3. 플레이 핵심

### 3-1. 핵심 플레이 루프
1. **탐색** — 마을/지역을 자유롭게 돌아다니며 환경 관찰
2. **대화** — NPC와 상호작용, 행동에 따라 반응 변화
3. **선택** — 스토리 분기점에서의 결정 (아이 거부/수용 등)
4. **전투** — 심플한 액션. 무기 휘두르기 중심, 스킬 시스템 없음

### 3-2. 전투 시스템
- **인카운터:** 심볼 인카운터 (필드에서 접촉 시 전투 씬 돌입)
- **전투 대상:** 전투 가능 오브젝트 전반 (몬스터, 적대 NPC 등)
- **전투 조작:** 무기 휘두르기 중심, 스킬 없음
- **세부 전투 규칙:** Phase 0 이후 별도 논의

### 3-3. 전투의 서사적 역할
- 폭력은 선택지이되, 올바른 해결책은 아님
- 전투만으로도 엔딩 도달 가능 → but 진정한 엔딩은 아님 (몰살 루트)
- 감정적 교류/비폭력적 접근이 진엔딩으로 이어짐
- "폭력으로 문제를 해결할 수 있는가?"에 대한 게임적 답변 제공
- Undertale의 철학 계승: 플레이어의 선택을 막지 않되, 그 뒷감당은 유저의 몫

### 3-4. 전투 빈도 — 환경 연동
- 초반(쌍눈박이 마을 도착): 차별주의자가 시비 → 전투 빈번
- 중반(교역 도시): 주인공의 연승으로 덤비는 자 감소 → 전투 줄어듦
- 환경과 플레이어의 행동/평판에 따라 동적으로 조절

## 4. 세계관

### 4-1. 기본 구도
- **외눈박이 마을:** 소수 집단. 세상 대부분은 쌍눈박이.
- **쌍눈박이 세상:** 다수 집단. 외눈박이를 기괴하게 여기며 차별하는 이들이 존재.
- 차별하지 않는 쌍눈박이가 더 많지만, 차별주의자가 플레이에 지장을 줄 정도로 등장.
- 선악의 구분이 아닌 **다름의 구도** — 같은 행동이 맥락에 따라 예의도, 무례도 됨.

### 4-2. 성인식 여정
- 외눈박이 마을에서 성인으로 인정받으려면 **마을 밖 세상을 경험하고 돌아와야 함**.
- 단순 통과의례가 아닌 **생존 전략** — 다수의 세계를 이해해야 공존할 수 있음.
- **동행 금기:** 같은 성인식 여정 중인 젊은이들과 조우는 가능하나, 함께 여행하는 것은 금기. 소수자인 자신에 대한 인식을 온몸으로 받아내야 하기 때문.
- 성인식의 진정한 의미: 단순히 바깥 세상을 보고 오는 것이 아니라, **차별에 대한 의문을 품고 돌아오는 것**.

### 4-3. 전설의 유물
- **효과:** 외눈박이는 쌍눈박이로, 쌍눈박이는 외눈박이로 **외견을 뒤바꿈**.
- **발동자 불변:** 유물을 발동시킨 사람만 외견이 변하지 않음 → 유일한 이방인이 됨.
- **영구적 효과:** 다시 발동되기까지 오랜 시간이 필요.
- **반복 발동의 역사:** 과거에도 계속 발동되어 왔으며, 모든 사람들은 자신의 본래 모습이 외눈박이인지 쌍눈박이인지조차 기억하지 못함.

### 4-4. 유물이 "전설"로만 남은 이유 (이중 은폐 구조)
- **1차 은폐 — 유물 자체의 속성:** 유물 발동 시 외견뿐 아니라 유물에 대한 기억도 희미하게 만듦. 시간이 흐르면 "원래부터 이랬다"고 믿게 됨. 유일하게 기억을 유지하는 건 발동자뿐.
- **2차 은폐 — 권력에 의한 조직적 은폐:** 쌍눈박이 권력자들이 다수자의 지위를 유지하기 위해 남은 기록마저 조직적으로 숨기거나 파괴. 외눈박이 마을의 구전은 소수 집단의 "미신" 정도로 취급하며 무시.
- **그럼에도 완전히 지워지지 않은 흔적들** → 교역 도시의 골동품/벽화로 남아있지만 아무도 연결 짓지 못함 → 이것이 복선의 근거.

### 4-5. 촌장직의 순환 구조
- 역대 촌장은 모두 과거 유물 발동자.
- 발동자는 영원한 이방인이 되지만, 마을로 돌아와 다음 세대에게 "전설"을 전함.
- **촌장의 자격은 바깥 세상을 바꿔본 자에게 주어짐.**
- 성인식 → 유물 발동 → 촌장이 되는 순환 구조.
- 촌장의 **의례용 반가면**: 역대 촌장에게 전해지는 전통 장식. 마을 사람들은 "촌장의 위엄을 나타내는 전통"으로 받아들이지만, 실은 유물 발동 후 바뀌지 않은 외견(쌍눈박이)을 감추기 위한 장치.

## 5. 스토리 구조 (챕터별)

### 챕터 0 — 외눈박이 마을 (프롤로그)
- 주인공의 일상, 마을 소개
- 가족, 이웃, 일상의 소소한 행복을 체험 → **이 마을이 얼마나 소중한 곳인지** 각인
- 성인식 여정 설명 — 동행 금기 규율
- 가면을 쓴 촌장이 유물의 존재를 넌지시 언급: "차별이 극에 달할 때, 이것이 세상을 뒤엎을 수 있을 것이다."
- **표면 복선:** 마을 내 쌍눈박이 예술품/골동품
- **플레이:** 탐색 + 대화 위주, 전투 없음. 평화로운 분위기.
- **감정:** 안정, 익숙함, 따뜻함

### 챕터 1 — 쌍눈박이 마을 (첫 접촉)
- 처음 접하는 쌍눈박이 세상
- 신기함 → 첫 차별 경험 → 점차 심화
- 차별주의자들이 시비 → **전투 빈번**
- **차별하지 않는 NPC들과의 진심 어린 교류** — 주인공을 도와주는 상인, 호기심 어린 사람들
  - 이 관계들이 쌓여야 → 나중에 유물 발동 후 "이 사람들도 피해를 받는다"는 무게가 생김
- 동료 젊은이와 첫 조우 — 공감과 위안
- **쌍눈박이 아이와의 만남** (거부/수용 시스템)
- **표면 복선:** 외눈박이용 골동품, "옛날엔 교류가 활발했다" 대사
- **플레이:** 탐색 + 대화 + 전투 (전투 비중 높음)
- **감정:** 이질감 → 차별 체험 → 분노 → 그 사이의 따뜻한 순간들

### 챕터 2 — 교역 도시 (공존과 균열)
- 외눈박이와 쌍눈박이가 공존하는 도시
- "여기서는 괜찮을 수도 있겠다"는 희망
- **다양한 관계의 스펙트럼:**
  - 사이 좋은 외눈박이-쌍눈박이 관계
  - 철천지원수인 관계
  - 쌍눈박이 간의 알력 다툼 (같은 편끼리도 갈라짐)
- 주인공의 연승으로 차별주의자들이 잘 덤비지 않음 → **전투 감소**
- 유물의 단서 발견 — 하지만 이 시점에서 **발동시킬 이유를 못 느낌** ("공존도 되는데 굳이?")
- **결정적 복선 집중 배치:** 고대 벽화, 대칭되는 전설, 과거 발동자 흔적, 촌장 가면과 유사한 유물
- 아이와의 관계 심화 (수용 시)
- 동료 젊은이와 재조우

- **챕터 2 후반 — 결정적 사건:**
  - 교역 도시의 공존이 **무너지는 사건** 발생
  - 차별주의 세력이 조직적으로 외눈박이를 추방하는 정책 시행
  - 주인공을 도와줬던 쌍눈박이 NPC가 "외눈박이 편"이라는 이유로 핍박
  - 동료 젊은이 중 하나가 심각한 피해 (부상, 투옥 등)
  - 아이(수용 시)도 위험에 처함 — 주인공과 함께 있다는 이유로
  - **공존이 가능하다고 믿었던 장소에서의 배신** → 유물 발동의 동기 생성

- **플레이:** 탐색 + 대화 중심, 전투 줄어듦 → 후반부 사건으로 긴장 고조
- **감정:** 희망 → 신뢰 → **배신감과 절망** → "이대로는 안 된다"

### 챕터 3 — 유물을 향한 여정 & 뒤집힘
- 결정적 사건 이후, 유물을 찾아 나섬
- **여정 자체가 갈등:**
  - 유물의 대가를 알게 됨 (발동자만 변하지 않는다)
  - 과거 발동자의 기록/흔적 발견 — 그들의 고독한 말년
  - "세상이 두 번이나 지우려 한 진실"을 발견
  - 동료와의 마지막 대화 — "네가 결정해"
  - 아이가 주인공의 손을 잡는 장면 (수용 시)
- **유물 발동 — 플레이어의 진심에서 나온 결단:**
  - "나 하나 희생해서 이 세상의 구조를 바꾸겠다"
  - 몰살 루트와의 대비: 폭력은 눈앞의 적만 처리하지만, 유물은 구조를 바꿈
- **발동 후 — 사회 혼란:**
  - 전 세계 외견 전환. 주인공만 외눈박이 그대로.
  - 갑자기 소수자가 된 전 다수파의 공포와 분노
  - 갑자기 다수가 된 전 소수파 중 일부의 복수심
  - 외눈박이 마을도 혼란 — "우리가 진짜 외눈박이인 건 맞아?"
  - 차별하지 않았던 사람들까지 피해를 입는 부조리
  - 교역 도시의 공존 질서 붕괴
- **플레이:** 여정 탐색 → 결단 → 혼란 속 탐색, 기존 NPC들의 변화된 반응 목격
- **감정:** 결의 → 희생적 결단 → 충격 → 혼란 → 고립감

### 에필로그 — 다시 외눈박이 마을 (충분한 호흡)
- 혼란이 어느 정도 가라앉은 후 마을로 귀환
- **마을을 다시 걸어다니는 플레이 구간:**
  - 쌍눈박이 외견이 된 마을 사람들의 혼란
  - 주인공을 경계하는 사람들
  - 챕터 0에서 소중했던 장소들을 다시 방문 → 변했지만 변하지 않은 것들
  - 가족이 "네 목소리잖아"라며 알아봐줌
  - 아이가 주인공 편에 서줌 (진엔딩)
  - **촌장이 반가면을 벗고 쌍눈을 드러냄 — "나도 그랬다"**
- **시간을 들여 "본질의 연결"을 체감하게 함**
- **감정:** 고립의 최저점 → 서서히 스며드는 따뜻함 → 깨달음과 감동

## 7. 핵심 캐릭터

### 7-0. 네이밍 체계

캐릭터 이름 자체가 세계관의 차별 구조를 반영하는 장치.

| 구분 | 이름 규칙 | 예시 |
|------|----------|------|
| **외눈박이** | 하나의 중심 모음 + 짧은 자음. 둥글고 단단한 한 덩어리. 1~2음절. | Nol, Cor, Mohl, Tun, Runo, Sols |
| **쌍눈박이** | 음절이 반복/대칭. 길고 장식적. "우리는 이름도 풍성하다"는 문화적 자부심. 2~3음절. | Lila, Miren, Seren, Ellem, Parin, Toriel |

- 발음하는 순간 구조 차이가 직관적으로 체감됨
- 차별 장면 활용: "Nol? 그게 이름의 전부야?" / 서류의 이름란 크기 차이
- 다수의 오만함이 Miren-Miren 같은 반복 이름을 짓는 문화를 만들어냈다는 풍자

### 7-1. 주인공
- 외눈박이 마을의 젊은이
- 성인식 여정을 떠남
- 유물 발동자 → 유일하게 외견이 변하지 않는 존재
- 외견은 절대 변하지 않음 (태생적인 것) — 변장 선택지는 존재 (가면, 낙서 등)

### 7-2. 루메 (Lume) — 쌍눈박이 아이

| 항목 | 내용 |
|------|------|
| 이름 | Lume (루메) — lumen(빛)에서 유래. "눈이 몇 개든 빛이 있어야 볼 수 있다" |
| 출신 | 쌍눈박이 마을 |
| 나이 | 7~9세 |
| 외견 | 쌍눈박이, 한쪽 눈 실명 (머리카락으로 가림 → 합류 후 주인공에게 안대를 선물받음) |
| 체형 | SD 2등신, 주인공보다 작고 동그란 체형 |
| 복장 | 낡고 커다란 물려받은/주운 옷. 쌍눈박이 마을 양식이나 해진 상태 |
| 성격 | 순수, 호기심 많음, 말수 적지만 핵심을 찌르는 한마디 |

**소지품:**
- **한쪽 눈이 빠진 낡은 동물 인형** — 유일한 소중품. 루메 자신의 투영. 3차 거부 시 버려진 채 발견.
- **주운 유리 조각** — 빛에 비추면 예쁘다며 보여줌. 호신용 무기 겸 장난감. lumen 모티브.

**주인공과의 상호작용 아이템:**
- **안대 (주인공 → 루메 선물)** — 합류 후 특정 시점에 선물. 머리카락 대신 안대 착용. 유대의 물리적 증거.
  - 3차 거부 루트: 안대를 받지 못한 채 떠남
  - 진엔딩: 안대를 쓴 루메가 주인공을 알아봄

**서사적 역할:**
- "다수에 속해도 안전하지 않다" — 차별은 눈의 개수 문제가 아님을 증명
- 주인공과의 대칭 — 어디에도 속하지 못하는 존재끼리의 연결
- 주인공에게 "네 눈, 예쁘다"고 말하는 최초의 존재
- 함께 다닐 때 고유 반응 (아이의 시선으로 세상을 비춤)
- **진엔딩 키 캐릭터**

### 7-3. 거부/수용 시스템
| 선택 | 결과 |
|------|------|
| 1차 수용 | 루메 합류 |
| 1차 거부 | 루메가 새끼오리처럼 따라옴 |
| 2차 수용 | 루메 합류 |
| 2차 거부 | 루메가 더 처량하게 따라옴 |
| 3차 수용 | 루메 합류 |
| 3차 거부 | 루메가 포기하고 떠남 → **진엔딩 잠김** |

- 3번 거부 후, 게임 후반부에 루메의 흔적(한쪽 눈 빠진 인형, 누군가의 언급)이 등장 → 2회차 동기부여

### 7-4. 동료 젊은이들
- 같은 성인식 여정 중인 외눈박이 젊은이들
- 특정 지점에서 조우 — 감상 공유, 공감 유도
- "나도 당했어" 식의 대화로 플레이어 감정 대변
- 챕터 2 후반 결정적 사건에서 심각한 피해를 입는 역할

### 7-5. 촌장
- **과거 유물 발동자** — 자신도 성인식 여정에서 유물을 발동시킨 경험이 있음
- 유물에 대한 이야기를 "전설"의 형태로 넌지시 전달 — 실은 자기 경험을 전설의 옷을 입혀 전하는 것
- **의례용 반가면:** 촌장직에 오르면 전해지는 전통 장식. 마을 사람들은 "촌장의 위엄"으로 받아들이지만, 실은 쌍눈박이 외견을 감추기 위한 장치.
- 에필로그에서 가면을 벗고 쌍눈을 드러내며 주인공에게 공감 — "나도 그랬다"
- **"그 자체로 충분하다"는 메시지를 이미 살아낸 사람**

## 8. 복선 설계

### 8-1. 배치 3원칙
1. **점진적 밀도** — 초반엔 희박하게, 후반으로 갈수록 밀도 증가. "뭔가 이상한데?"를 느끼기 시작하는 시점은 Ch2.
2. **이중 해석** — 모든 복선은 1회차에서 "세계관 디테일"로, 2회차에서 "이게 이거였어?!"로 읽혀야 함.
3. **감각 분산** — 시각(배경 오브젝트), 텍스트(NPC 대사), 상호작용(조사 가능 오브젝트)을 고르게 활용.

### 8-2. 복선 소재 분류

**표면 복선 (교류의 흔적으로 위장)**
- 외눈박이 마을의 쌍눈박이 그림/예술품
- 쌍눈박이 마을의 외눈박이용 골동품
- "옛날엔 교류가 활발했다더라" 식의 NPC 대사

**결정적 복선 (예리한 플레이어가 눈치챌 수 있는 것)**
- 촌장의 오래된 일기: "우리가 처음부터 이랬을까?"
- 쌍눈박이 마을의 고대 벽화: 유물과 함께 눈이 바뀌는 사람들 묘사
- 양쪽 마을에 전해지는 전설이 거울처럼 대칭 — 둘 다 "원래는 우리가 다수였다"는 이야기를 보유
- 유물 근처 기록: 발동 횟수를 표시한 듯한 눈금 자국
- 과거 유물 발동자들의 흔적 — 역사 속 영원한 이방인들
- 교역 도시에서 촌장의 가면과 유사한 유물/그림 발견
- 촌장의 외견에 대한 미묘한 힌트 (프롤로그) — 첫 플레이에선 눈치 못 챔, 2회차에서 소름

### 8-3. 챕터별 복선 배치 계획

**Ch0 — 외눈박이 마을 (씨앗 뿌리기, 2~3개)**
- 🎨 마을 광장에 쌍눈박이 문양이 새겨진 오래된 우물/석상 → 조사 시 "옛날 것이라 아무도 신경 안 쓴다"
- 💬 촌장의 유물 언급: 전설의 형태로, 구체적이지 않게
- 🎨 촌장의 반가면 — 디자인이 또렷하게 보이도록 연출하되 설명은 "전통"으로만

**Ch1 — 쌍눈박이 마을 (흩뿌리기, 3~4개)**
- 🎨 골동품 가게에 외눈박이용 도구/장신구 → 상인 대사: "수집가들이 좋아해. 옛날엔 교류가 있었나 봐"
- 💬 노인 NPC: "우리 할머니는 '원래 우리도 하나였다'는 옛날이야기를 하셨지" (표면: 교류 / 진실: 유물 발동 후 기억 변형)
- 🔍 도서관/기록보관소에서 찢긴 고서 한 페이지 — "눈이 바뀌는 사람들" 삽화 (조사 가능 오브젝트, 텍스트는 판독 불가)
- 💬 차별하지 않는 NPC: "한쪽 눈이면 불편하지 않아?" → 선의의 무례 (주제 강화)

**Ch2 — 교역 도시 (결정적 복선 집중, 4~5개)**
- 🎨 고대 벽화: 유물을 중심으로 눈이 바뀌는 사람들 → 관광 명소로 "신화적 예술품" 취급
- 🔍 골동품 가게에서 촌장 가면과 유사한 가면 발견 → 점주: "이런 건 여러 도시에서 나와. 옛 지도자들의 공통 양식인가 봐"
- 💬 양쪽 전설의 대칭 발견: 외눈박이 구전 "원래 다 외눈이었다" vs 쌍눈박이 기록 "원래 다 쌍눈이었다" → 동시에 접하면 모순 인지
- 🔍 뒷골목 서고에서 과거 발동자의 일지 단편 → "나만 바뀌지 않았다. 모두가 나를 이상하게 본다."
- 💬 촌장과 같은 세대로 보이는 노인이 "저 가면... 어디서 봤는데" 하고 흐지부지

**Ch3 — 유물 여정 (퍼즐 완성, 3~4개)**
- 🔍 유물 근처 벽면의 눈금 자국 — 발동 횟수 기록
- 🔍 과거 발동자들의 기록 모음 — 기억의 조각과 연동
- 💬 유물의 기억 소거 효과를 직접 체감하는 연출 (NPC가 방금 들은 얘기를 잊는 장면)
- 🎨 유물 주변 환경이 Ch0 마을 석상과 동일한 문양

## 10. 다회차 요소

### 10-1. 엔딩 분기
| 엔딩 | 조건 | 톤 |
|------|------|-----|
| **진엔딩** | 아이 수용 + 비폭력 중심 + 유물 발동 | 따뜻한 깨달음 |
| **일반 엔딩** | 아이 거부 + 유물 발동 | 쓸쓸한 귀환 |
| **몰살 엔딩** | 전투로 모든 것을 해결 | 어둡고 공허한 결말 |
| **회피 엔딩** | 유물을 발동시키지 않고 귀환 | 아무것도 변하지 않은 세상 |
| **히든 엔딩** | 기억의 조각 전부 수집 | 세계관의 진실 |

### 10-2. 기억의 조각 (엔딩별 획득 수집 아이템)
- 각 엔딩 도달 시 해당 엔딩과 공명하는 조각 1개 획득
- 과거 유물 발동자들이 남긴 기록 파편 — 세계관의 진실을 밝히는 복선 기능

| 엔딩 | 획득 조각 | 밝혀지는 진실 |
|------|----------|-------------|
| 진엔딩 | 조각 A | 최초 발동자의 동기 — "차별을 끝내고 싶었다" |
| 일반 엔딩 | 조각 B | 유물의 기억 소거 원리 — 왜 모두가 잊는가 |
| 몰살 엔딩 | 조각 C | 폭력으로 해결하려 했던 과거 발동자의 실패담 |
| 회피 엔딩 | 조각 D | 유물을 외면한 자들의 기록 — "아무것도 바뀌지 않았다" |
| 히든 엔딩 | 조각 E (최종) | 유물의 기원 / 외눈-쌍눈의 본래 모습에 대한 답 |

- 조각 A~D 전부 수집 시 히든 엔딩 개방 → 조각 E 획득
- 각 조각이 해당 엔딩의 선택과 공명 → 플레이어의 선택을 되돌아보게 함

### 10-3. 히든 엔딩
- 기억의 조각 A~D 전부 수집 시 개방
- 유물의 진짜 비밀 / 순환 자체를 끊는 제3의 선택 등
- 세계관의 전모를 파악할 수 있는 루트

### 10-4. 회차별 차별화 설계

**"같은 게임이지만 완전히 다른 경험"** — 반복감 제거를 위한 5개 레이어:

**레이어 1: NPC 반응 체계 변화**
| 루트 | NPC 반응 |
|------|---------|
| 진엔딩 | 점차 마음을 열어감. 따뜻한 대사, 도움, 숨겨진 이야기 |
| 일반 | 중립~쓸쓸. 아이 부재로 특정 이벤트 미발생 |
| 몰살 | 공포/적대. 거리가 비고, 문이 잠기고, NPC 도주 |
| 회피 | 무관심. 주인공을 그냥 지나가는 여행자로 취급 |

**레이어 2: 환경/맵 변화**
- 몰살 루트: 파괴 흔적, 폐허, 핏자국 암시
- 회피 루트: 모든 게 그대로 — "변하지 않음"의 공허함
- 진엔딩 2회차 이후: 1회차에서 못 들어갔던 장소 개방 (촌장 서재, 유물 관리실 등)

**레이어 3: Ch3 내용 분기**
- 유물 여정(진/일반): 탐색/퍼즐, 과거 발동자 추적
- 몰살: 전투 연속, 적대적 세계
- 회피: 짧은 귀환길, 텅 빈 에필로그

**레이어 4: 아이의 존재감**
- 있을 때: 아이의 반응, 대화, 아이를 지키려는 긴장감. 아이만 발견하는 것들.
- 없을 때: 빈 자리. 아이의 흔적(버려진 장난감, "여기 아이가 있었는데...") → 부재의 무게.

**레이어 5: 기억의 조각 — 메타 시점**
- 이전 회차의 조각 지식이 플레이어의 해석을 바꿈
- 같은 NPC 대사라도 "아, 이게 그 뜻이었구나"
- 게임 내 강제 연출 없이 플레이어 스스로 다르게 느끼는 구조

### 10-5. 주회차 연동 연출 (후순위)
- 2회차 시작 시 촌장이 플레이어를 알아보는 듯한 대사
- 몰살 루트 이후 일부 NPC의 미묘하게 다른 반응
- 현재 단계에서는 후순위이나, NPC 대사 테이블 설계 시 회차 변수를 구조적으로 포함시킬 것

### 10-6. 플레이 사이클 (5회차, 전 엔딩 플레이)

| 회차 | 내용 | 예상 시간 |
|------|------|----------|
| 1회차 | 스토리 체험 (진엔딩 or 일반 엔딩) | 3~4시간 |
| 2회차 | 다른 선택지 (일반 or 진엔딩) | 2~3시간 |
| 3회차 | 몰살 루트 (전투 중심, 빠른 진행) | 2~3시간 |
| 4회차 | 회피 루트 (가장 짧은 루트) | 1~1.5시간 |
| 5회차 | 조각 A~D 수집 완료 → 히든 엔딩 | 1~2시간 |
| **총합** | | **~10-12시간** |
